Documentation. 

What kind you need depends on who you are and where you are coming from.

  • The API Explorer is designed as an aid to explore the RESTful web API to SWAPP. It allows a developer to issue PUT, POST, DELETE and GET requests to a running instance of the SWAPP server and to inspect the responses. Thus, the API Explorer provides “live” documentation of SWAPP’s web API and works as a combination between documentation and a development tools. SWAPP currently allows you to explore the following URIs:
    • /swapp/login
    • /swapp/admin/*
    • /swapp/rdf/db
    • /swapp/session/db
    • /swapp/session/eq
