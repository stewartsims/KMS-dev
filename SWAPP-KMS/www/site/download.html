Download. 

You may download SWAPP from GIT. You also need a working installation of SWI-Prolog.

  • http://prolog.cs.vu.nl/git/contrib/SWAPP.git
    Browser access; allows viewing history, files and download snapshots
  • git://www.swi-prolog.org:/home/pl/git/contrib/SWAPP.git
    Clone-able GIT repository, native GIT protocol
  • http://www.swi-prolog.org/Download.html
    SWI-Prolog download page
